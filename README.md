# Notelite

Notelite is a mini Notion-style notes app built for a workshop demo. It uses React, Vite, Tailwind CSS, JavaScript, and `localStorage`.

## Features

- Dark-mode-only workspace
- Sidebar with pages, search, pinned notes, and active note highlighting
- Create, duplicate, pin, and delete notes
- Editable note icon, title, and content
- Autosave to `localStorage`
- Last edited time
- Markdown preview toggle
- Keyboard shortcut: `Ctrl + N` or `Cmd + N` creates a new note
- Responsive layout

## Setup

```bash
npm install
npm run dev
```

Then open the local Vite URL shown in your terminal.

## Build

```bash
npm run build
```
